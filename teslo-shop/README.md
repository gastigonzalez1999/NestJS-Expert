# Teslo API

1. Clone the project
2. ``` yarn install ```
3. Use ``` .env template ``` as ``` .env ```
4. Build db
``` docker-compose up -d ```
5. Execute Seed
``` localhost:3001/api/seed ```
6. ``` yarn start:dev ```
